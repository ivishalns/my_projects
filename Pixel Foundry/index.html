Token Architecture
Primitives
primitives.json
Raw brand values: 47 colors, 6 fonts, spacing grid, breakpoints. Swap to rebrand.
Semantic Tokens
tokens.css
150+ CSS custom properties. Surface, text, border, interactive, status, data viz. Light/dark mode.
Component Tokens
tokens.css
Per-component specs: button sizes by density, table row heights, touch targets, chart colors.
shadcn Bridge
shadcn-bridge.css
Remaps shadcn/ui CSS variables to Deloitte tokens. Includes 8 animations, 12 utility classes.
Tailwind Override
tailwind.config.ts
Forces Tailwind defaults to Deloitte values: text-sm=13px (not 14px), removes rounded-2xl/3xl.
