Guidelines.md
Three tiers. Priority-ordered.
Zero credits to deploy.
The silent enforcer. AI reads it before every prompt. Tier 1 at the top = highest priority.
Tier 1 — Structural rules · Highest priority

Prevents layout & component breakage

8px spacing grid • Flexbox/grid default • Component height locks • Text overflow rules • Z-index hierarchy • Anti-overlap enforcement • State coverage requirements • Anti-pattern constraints

Tier 2 — Use-case specific · Per project

Domain intelligence from intake

Industry terminology • Data format rules • Compliance requirements • Persona context • Domain-specific component rules

Tier 3 — Design tokens · From Phase 5

Visual language every screen inherits

Typography specs • Color palette with hex values • Component styles • Modern design vocabulary • 60-30-10 color rule

Context Ledger
Zero context loss across 8 phases.
